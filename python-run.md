# Python 代码运行器 ⚡

在线运行 Python 代码，快速、简洁、无卡顿。

<PythonEditorLite />

## 💡 使用说明

1. 在上方编辑器中输入 Python 代码
2. 点击 **"▶️ 运行代码"** 或按 `Ctrl/Cmd + Enter`
3. 查看下方的输出结果

## ⌨️ 快捷键

- **Ctrl/Cmd + Enter**: 运行代码
- **Tab**: 插入 4 个空格

## 📋 示例代码

### Hello World
```python
print("Hello, FastAPI!")
```

### 计算
```python
x = 10
y = 20
print(f"{x} + {y} = {x + y}")
```

### 列表处理
```python
numbers = [1, 2, 3, 4, 5]
print("数字:", numbers)
print("总和:", sum(numbers))
print("平均:", sum(numbers) / len(numbers))
```

### 循环
```python
for i in range(5):
    print(f"第 {i+1} 次循环")
```
